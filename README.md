### Sublime Text 3 C++ Build System

##### How to install?
----------------------------------------------------------------------------------------------------------------------------
  - Tools -> Build System -> New build system...
  - Pase the content above
  - Save as C++ Build and Run.sublime-build  
  
  
##### How to use?
------------------------------------------------------------------------------------------------------------------------------- 
  - Open a C++ file
  - Select Tools -> Build System -> Build and Run
  - cmd + b
